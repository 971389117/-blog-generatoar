获取元素的属性

元素.attributes返回元素对象的所有属性，它是一个实时的array like 动态对象
它的每一个元素也是一个属性对象
planA：attributes 

            $('#idP').attributes
            $('#idP').attributes[0]
            $('#idP').attributes.id.value
            $('#idP').attributes['id'].name  //同 nodeName / nodeValue
planB：Element.标准属性 它获得的是属性值 
            $('#idP').id // ->string
            $('#idP').htmlFor  // 这个是假的
            //获取class 的字符串形式 形如："classP classB"
            $('#idP').className
getAttribute 它和planB 的返回值一样
planC：元素.getAttribute(属性名) 
            $('#idP').getAttribute('id')
获得所有属性名字，帮助获得属性值
元素.getAttributeNames() ：Array
判断是否有属性
元素.hasAttribute(属性名)
元素..hasAttributes()
