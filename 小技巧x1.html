in 和 for in 运算符

in
继承的属性也会返回 true
for in
遍历所有 enumerable:true 的属性
            场景 1：遍历自身所有可遍历的属性
            function Animal(name){
                this.name=name;
            }
            function Cat(age){
                this.age=age
            }
            Cat.prototype=new Animal('猫咪')
            let cat = new Cat(18)

            for(let i in cat){
                if(cat.hasOwnProperty(i)){
                    console.log(i)
                }
            }
